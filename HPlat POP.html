Development

This will be an overview of my development as a coder. Starting from humble beginnings (you know nothing, John Snow) i've recently (dec'17) started learning HTML, CSS and JavaScript. Still in it's infant stages, but already crawling about. I hope to master these 3 languages in 2018, and while I'm at it, I'll be adding PHP and possibly JAVA into the mix.

Learning so far

Codecademy: a website for the utmost beginner such as myself. You'll learn the basics of many languages, through reading about, and playing with code. Mainly the playful aspect of learning helps me to stay motivated. Coding can be fun! Current progress: introduction to HTML, introduction to CSS, introduction to JavaScript

Sololearn: like codecademy, this is also a website for beginners. Likewise, a lot of reading, mixed with exercizes. Great site to learn the basics too. I recommend using both sites, as they have slightly different takes on things, which will help you get a more rounded introduction. Current progress: introduction to HTML, introduction to CSS, introduction to JavaScript

CodeGorilla: I've applied for a traineeship (or "bootcamp") at CodeGorilla. They seem to be a great team of people, that have built their company around the fact that there's a large shortage of programmers, and also many people that might want to make the shift into IT but don't know where to start. They offer a bootcamp of 4 months, where you'll learn in-depth PHP, both for frontend and backend development. To get into the bootcamp there is first a full month of studies at home, plus a "hackathon" on the 6th and 7th of january 2018. I'm stoked! During the home studies, we were required to build 2 things: an addressbook using HTML, CSS & JavaScript, and this site (also using HTML, CSS & JavaScript).
